# bash-asdf

a script to record notes and output timecoded files

## References
* http://askubuntu.com/questions/484718/how-to-make-a-file-executable

#### Copy asdf to ~
        cp asdf ~
#### Create the asdf.d directory
        mkdir ~/asdf.d
#### Make asdf executable
        chmod +x asdf
#### Run the script
        ./asdf
